Anteater Network

Description

Anteater Network is a new, clean, and expandable system created for the UCI Alumni Association as my senior design project for the informatics undegraduate program.

The system helps connect current and past students and faculty with other alumni who have started their own businesses by displaying these businesses on a map. This allows for a highly effective visual representation of the spread of UCI's alumni network.

Features

  • Displays all UCI alumni owned businesses
  • Supports online businesses
  • Filter by city, business name, and zip code
  • Search by business name and zip code
  • Auto focusing map on search, with buffer zones around pins
  • Auto populating menus
  • Hide-able results list
  • Marker clustering

Alien Escape

Description

Alien Escape is a short, unique video game as part of a 7 week project involving creating prototypes, presentations, and documentation. It is an original, tile based, adventure game built on the melonJS HTML5 game library and Tiled map format. It features custom artwork and sprites, music and sound effects, and artificial intelligence and field of view algorithms.
