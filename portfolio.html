Anteater Network

Description

Anteater Network is a new, clean, and expandable system created for the UCI Alumni Association as my senior design project for the informatics undergraduate program.

The system helps connect current and past students and faculty with other alumni who have started their own businesses by displaying these businesses on a map. This allows for a highly effective visual representation of the spread of UCI's alumni network.

The project was developed in an agile environment with continuous, iterative cycles of development, presentations, and feedback. The design process involved creating prototypes, wireframes, use case diagrams, architectural diagrams, project plans, test plans, and requirements documents. Testing included JavaScript unit testing, execution tests, code reviews, user testing, heuristic evaluations, and expert reviews.

Features

  • Displays all UCI alumni owned businesses
  • Supports online businesses
  • Filter by city, business name, and zip code
  • Instantly search by business name and zip code
  • Auto focusing map on search, with buffer zones around pins
  • Auto populating menus
  • Hide-able results list
  • Marker clustering

Alien Escape

Description

Alien Escape is a short, unique video game as part of a 7 week project involving creating prototypes, presentations, and documentation. It is an original, tile based, adventure game built on the melonJS HTML5 game library and Tiled map format. It features custom artwork and sprites, sounds, and artificial intelligence and field of view algorithms.

Features

  • Built using the melonJS game library
  • 3 challenging levels, created with Tiled Map Editor
  • Original sprite based artwork
  • Original music and sound effects, triggered by events and sound zones
  • Custom fog of war algorithm
  • 4 different types of enemy AI: Novice, Pro, Elite, Doctor
  • 2 different types of turret AI: Normal, Advanced

Terris Minecraft Community

Description

Terris is a gaming community that I co-founded in December 2011. The community is largely focused on playing Minecraft, though it has been steadily expanding to other games such as Team Fortress 2. I resigned from my role as a staff member in September 2012 to focus on other hobbies, though I occasionally help out when needed.

At that time, I managed a team of twenty staff along with a community of a few hundred registered users and actively created, managed, and led new projects for the community. Managed and maintained all aspects of the community including the finances, social media presence, graphical identity, technical support, and in game operations.

Features

  • Served as the main technical support, set up and managed a CentOS 5.8 Linux dedicated server
  • Designed graphical identity for the entire community
  • Managed a team of twenty staff members
  • Hosted weekly staff meetings via Skype
  • Provided guidance to staff and players in accordance with the community's vision and values
  • Constantly looked for ways to expand the community
  • Created and led new projects, contests, and raffles for the community
